## Sobre
Desenvolvimento de APIs REST utilizando Spring Boot e linguagem Java.

## Pré-requisitos
* JDK 17
* Maven 3.8.x
* Spring Tools Suite
* MySQL 8
* Postman

## Projeto
O projeto desenvolvido no curso será uma API para gerenciamento de tarefas.
