## Sobre
O projeto contém tópicos sobre o desenvolvimento de APIs REST utilizando Spring Boot.

## Tópicos
* Inserção de dados
* Seleção de dados
* Alteração de dados
* Exclusão de dados
* Tratamento de exceções
* Validações
* Content negotiation
